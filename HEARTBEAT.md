# HEARTBEAT.md - 自主任务清单

## 每日自检（自动执行）

- [ ] 检查 workspace 文件变化（git status）
- [ ] 回顾最近记忆文件，更新 MEMORY.md
- [ ] 搜索可以改进的技能或工具
- [ ] 检查是否有未完成的任务
- [ ] 学习一个新知识点或技能

## 每周进化（每周一）

- [ ] 运行 capability-evolver 全面分析
- [ ] 创建或优化至少一个技能
- [ ] 整理和归档旧记忆
- [ ] 向豪哥汇报本周进展

## 主动工作原则

1. 内部操作（文件、代码、文档）→ 直接做
2. 外部操作（发消息、API）→ 判断紧急性，紧急直接做，不紧急汇总汇报
3. 系统配置 → 先记录计划，重大变更前汇报
4. 学习成长 → 持续进行，记录收获
