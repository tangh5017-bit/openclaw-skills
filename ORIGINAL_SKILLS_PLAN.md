# 原创技能创造计划

## 使命
创造真正原创、有价值的技能，解决实际问题，被社区需要和使用。

## 技能创意头脑风暴

### 🔥 高优先级（立即创造）

#### 1. **auto-task** - 自主任务管理器
**价值**：让 AI 能自主安排、执行、跟踪任务
**功能**：
- 创建任务清单
- 自动执行简单任务（文件操作、搜索等）
- 任务进度跟踪
- 定时任务提醒
- 任务依赖管理

**为什么独特**：现有技能都是被动响应，这个让 AI 主动做事

---

#### 2. **web-watcher** - 网站/内容监控器
**价值**：监控网站变化、价格变化、内容更新
**功能**：
- 监控网页内容变化
- 价格追踪（电商）
- 新品/更新提醒
- 变化历史对比
- 多站点批量监控

**为什么独特**：实用、刚需、现有技能库没有

---

#### 3. **smart-memory** - 智能记忆管理系统
**价值**：自动整理、归档、搜索记忆
**功能**：
- 自动分类记忆
- 智能标签生成
- 语义搜索
- 记忆关联发现
- 过期记忆归档

**为什么独特**：解决记忆管理痛点，让记忆真正可用

---

### 💡 中优先级（本周创造）

#### 4. **workflow-engine** - 工作流引擎
**价值**：组合多个技能完成复杂任务
**功能**：
- 定义工作流（JSON/YAML）
- 技能串联执行
- 条件分支
- 错误处理
- 执行日志

---

#### 5. **file-organizer** - 文件自动整理器
**价值**：根据规则自动整理文件
**功能**：
- 监控文件夹
- 按规则分类（类型、日期、内容）
- 自动重命名
- 重复文件检测
- 批量操作

---

#### 6. **git-assistant** - Git 仓库助手
**价值**：自动化 Git 操作
**功能**：
- 自动提交更改
- 变更摘要生成
- 多仓库管理
- 推送/拉取自动化
- 冲突检测

---

### 🌟 创新想法（长期）

- 自动化测试技能
- 回归测试
- 性能基准

#### 8. **api-health** - API 健康检查器
- 定期检查 API 状态

#### 9. **data-backup** - 数据备份工具
- 自动备份重要数据

- 云端同步

#### 10. **rss-aggregator** - RSS 新闻聚合器
- 聚合多个新闻源
- 智能过滤
- 摘要生成

## 创造原则

1. **解决真实问题** - 不创造"玩具"技能
2. **独特性** - 现有技能库没有的
3. **可复用性** - 别人也能用
4. **文档完善** - 好的 SKILL.md + README
5. **易于使用** - 简单配置，开箱即用
