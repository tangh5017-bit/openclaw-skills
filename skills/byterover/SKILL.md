# ByteRover Skill

Comprehensive file system exploration, analysis, and management capabilities.

## Features
- Deep file system traversal and mapping
- File type detection and content analysis
- Directory structure visualization
- Large file handling and chunking
- Binary file inspection and metadata extraction
- File permission and ownership analysis
- Disk usage optimization suggestions

## Usage
- `/byterover explore <path>` - Explore directory structure
- `/byterover analyze <file>` - Analyze file content and metadata
- `/byterover search <pattern>` - Search files by pattern
- `/byterover visualize <path>` - Create directory visualization
- `/byterover optimize <path>` - Suggest storage optimizations
- `/byterover backup <path>` - Generate backup strategies
- `/byterover monitor <path>` - Monitor file changes

## Requirements
- Uses built-in file system tools
- No external dependencies or API keys
- Works with all file types and permissions
- Efficient handling of large directories and files
