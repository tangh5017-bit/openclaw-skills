# Find Skills

Search for and discover skills in your OpenClaw workspace and system.

## Usage

- `/find-skills query` - Search for skills matching the query
- `/find-skills list` - List all available skills
- `/find-skills info <skill-name>` - Get detailed information about a specific skill

## Features

- Searches through workspace skills, managed skills, and bundled skills
- Provides skill metadata, description, and usage examples
- Helps discover new capabilities and find the right tool for your task
