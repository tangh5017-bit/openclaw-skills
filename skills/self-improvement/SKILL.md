# Self Improvement Skill

Provide personalized self-improvement guidance, habit tracking, and goal setting strategies.

## Features
- Daily habit suggestions
- Goal setting frameworks (SMART goals, OKRs)
- Productivity techniques (Pomodoro, time blocking)
- Mindfulness and meditation guidance
- Learning strategy recommendations
- Progress tracking templates

## Usage
- `/self-improvement habits` - Get daily habit suggestions
- `/self-improvement goals` - Set and track goals
- `/self-improvement productivity` - Get productivity tips
- `/self-improvement learning` - Get learning strategies
- `/self-improvement mindfulness` - Get mindfulness exercises

## Requirements
- No external dependencies
- Works entirely with built-in AI capabilities
- Personalized based on user context and preferences
